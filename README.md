# TRAINABULL

**Training Bulls for the age of AI.**

A USF community advancing **clinical informatics and artificial intelligence** in healthcare — bridging medicine, nursing, engineering, and information sciences.

This repo is both the public landing page and the platform for the lab. Anyone with write access can update content.

## Live site

[https://taiyohanna.github.io/trainabull](https://taiyohanna.github.io/trainabull) *(after GitHub Pages is enabled — see below)*

## How to edit content (no command line needed)

1. Open the file you want to change in this repo on GitHub
2. Click the pencil icon (top right)
3. Make your edit
4. Scroll down → **Commit changes**
5. The site rebuilds automatically within ~1 minute

That's it. No Jekyll, no build step, no deploy — just plain HTML with Tailwind via CDN.

## How to add a collaborator

Repo settings → Collaborators → Add people. Once added, they can edit any file directly from the GitHub web UI.

## How to enable GitHub Pages (one-time setup)

1. Repo settings → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** · Folder: **/ (root)** · Save
4. Wait ~1 min for the first build
5. Site live at `https://taiyohanna.github.io/trainabull`
